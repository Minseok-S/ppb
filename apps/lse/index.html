사용방법

개인정보처리방침·동의서 등 HTML 문서를 불러와 직접 편집하고 HTML로 저장하는 도구입니다.

1

파일 불러오기

상단 파일 선택 버튼을 클릭하거나, 왼쪽 화면에 HTML 파일을 드래그&드롭합니다.

2

텍스트 직접 편집

오른쪽 편집 화면에서 원하는 곳을 클릭하면 바로 텍스트를 수정할 수 있습니다. 회사명, 담당자, 연락처, 조항 내용 등을 자유롭게 변경하세요.

3

서식 적용

텍스트를 드래그하여 선택한 후 툴바에서 굵기·기울임·글자 크기·색상·정렬 등을 적용합니다. 단락 형식으로 제목(H1~H3) 또는 본문을 지정할 수도 있습니다.

4

표(Table) 삽입 · 편집

툴바의 표 아이콘(⊞)을 클릭하면 격자가 나타납니다. 원하는 행·열 수에 마우스를 올리고 클릭하면 표가 삽입됩니다.
표 셀 위에서 우클릭하면 행·열 추가/삭제 메뉴가 표시됩니다.

5

링크 삽입

링크를 걸 텍스트를 선택한 후 툴바의 링크 아이콘(🔗)을 클릭합니다. URL 입력창이 열리면 주소를 입력하고 삽입합니다.

6

변경사항 비교

수정된 부분은 빨간 테두리로 강조됩니다. 왼쪽의 원본과 비교하며 확인하세요. 상단 원본 숨기기 버튼으로 편집 공간을 넓힐 수 있습니다.

7

HTML 내보내기

편집 완료 후 상단 HTML 내보내기 버튼을 클릭하면 원본 CSS·스타일이 유지된 HTML 파일이 다운로드됩니다.
